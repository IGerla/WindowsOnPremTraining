# Modul 1 – Erste VM erstellen

## Lernziele

Nach diesem Modul kannst du:

- Eine neue VM in Hyper-V erstellen (Generation 2)
- Windows Server 2022 von einer ISO-Datei installieren
- Die wichtigsten VM-Einstellungen (RAM, CPU, Festplatte, Netzwerk) erklären
- Einen Checkpoint (Snapshot) erstellen und dessen Nutzen beschreiben

---

## Hintergrund: Virtuelle Maschinen

Eine VM ist wie ein Computer im Computer. Du erstellst eine "leere Kiste" (VM), gibst ihr Hardware (RAM, CPU, Festplatte) und installierst ein Betriebssystem – genau wie bei einem physischen Server, nur ohne Schraubenzieher.

| Physischer Server | Virtuelle Maschine |
|---|---|
| RAM-Riegel einbauen | RAM-Wert in Einstellungen setzen |
| Festplatte einbauen | Virtuelle Festplatte (VHDX) erstellen |
| DVD einlegen | ISO-Datei zuweisen |
| Netzwerkkabel einstecken | Virtuelles Switch zuweisen |
