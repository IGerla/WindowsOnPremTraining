# Modul 2 – Grundkonfiguration

## Lernziele

Nach diesem Modul kannst du:

- Den Servernamen (Hostname) ändern
- Eine statische IP-Adresse konfigurieren
- Windows Updates manuell prüfen und installieren
- Den Server Manager als zentrale Verwaltungskonsole nutzen
- Grundlegende PowerShell-Befehle für die Serverkonfiguration anwenden

---

## Hintergrund: Warum konfigurieren?

Ein frisch installierter Windows Server hat einen zufälligen Namen (z.B. `WIN-A8F3KD92`) und bezieht seine IP-Adresse automatisch per DHCP. Für einen Server ist das unpraktisch:

- Andere Computer müssen ihn zuverlässig finden → **fester Name** nötig
- Dienste müssen immer unter derselben Adresse erreichbar sein → **feste IP** nötig
- Sicherheitslücken müssen geschlossen werden → **Updates** nötig
